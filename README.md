# Odin Sign-Up Form

A desktop-only sign-up page built using **HTML**, **CSS**, and **JavaScript** as part of **The Odin Project** curriculum.

The project recreates the provided sign-up page design while implementing semantic HTML, modern CSS practices, and client-side form validation.

---

## Preview

The webpage consists of:

- A left sidebar containing a placeholder for the background image.
- A branding/logo section.
- A right section containing introductory text.
- A sign-up form with validation.
- A "Create Account" button and login link.

## Validation Rules

The form validates the following fields:

### First Name

- Cannot be empty

### Last Name

- Cannot be empty

### Email

- Must be in a valid email format

Example:

```
john@example.com
```

### Phone Number

- Must contain 10–15 digits
- Allows:
  - Numbers
  - Spaces
  - '+' sign
  - Hyphens

### Password

- Minimum 8 characters

### Confirm Password

- Must exactly match the password
